# kk

> Минималистичный консольный текстовый редактор для UNIX-подобных систем, написанный на Rust.

![Rust](https://img.shields.io/badge/Rust-stable-orange?logo=rust)
![Platform](https://img.shields.io/badge/platform-UNIX-lightgrey)
![License](https://img.shields.io/badge/license-MIT-blue)

---

## Возможности

- Редактирование текстовых файлов прямо в терминале
- Встроенный файловый менеджер для навигации по директориям
- Поддержка кириллицы и Unicode
- Подсветка синтаксиса большинства языков программирования
- Отображение статуса изменений (`[modified]`)
- Минимальные зависимости, максимальная скорость

---

## Установка

```bash
git clone https://github.com/kempedron/kk
cd kk
cargo build --release
sudo cp target/release/kk /usr/bin
```

---

## Использование

```bash
kk <файл>      # открыть файл для редактирования
kk <папка>     # открыть файловый менеджер
```

---

## Горячие клавиши

### Редактор

| Клавиша  | Действие                        |
|----------|---------------------------------|
| `Ctrl+S` | Сохранить файл                  |
| `Ctrl+W` | Сохранить файл и выйти          |
| `Ctrl+Q` | Выйти (с запросом на сохранение)|
| `↑ ↓ ← →`| Навигация по тексту             |
| `Enter`  | Новая строка                    |
| `Backspace` | Удалить символ               |

### Файловый менеджер

| Клавиша  | Действие                        |
|----------|---------------------------------|
| `↑ ↓`   | Навигация по файлам             |
| `Enter`  | Открыть файл / войти в папку    |
| `Ctrl+D` | Перейти в родительскую папку    |
| `q`      | Выйти                           |
